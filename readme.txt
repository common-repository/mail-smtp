== Description ==

Simply configure your SMTP server and send email directly from WordPress site. This plugin allows you to easily send email through SMTP server and phpmailer without modifying the code and it works with most of the available themes. Just install and activate this plugin to use SMTP server and send emails in WordPress.<br><br>Read more about this plugin on <a href="https://siriuspro.pl">WordPress</a> blog.

== Installation ==

1. Unzip the mail-smtp.zip folder.
2. Upload the mail-smtp folder to your /wp-content/plugins directory.
3. In your WordPress dashboard, head over to the Plugins section.
4. Activate Mail SMTP plugin.
